# Modern Music Player

A beautiful and feature-rich music player built with HTML, CSS, and JavaScript that supports multiple audio formats.

## Features

- Support for multiple audio formats (MP3, WAV, AAC, MP4, and more)
- Modern and responsive user interface
- Playlist management
- Play, pause, next, and previous track controls
- Progress bar with seek functionality
- Volume control
- Time display (current time and duration)
- Drag and drop file support
- Beautiful animations and transitions

## Supported Audio Formats

The music player supports all audio formats that are supported by the HTML5 Audio API, including:
- MP3
- WAV
- AAC
- MP4 (audio)
- OGG
- WEBM
- FLAC (browser dependent)

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Click the "Add Music" button to select audio files from your computer
4. Your music will be added to the playlist
5. Click on any song in the playlist to play it
6. Use the controls at the bottom to:
   - Play/Pause the current track
   - Skip to next or previous track
   - Adjust volume
   - Seek through the current track using the progress bar

## Controls

- **Play/Pause Button**: Toggle playback of the current track
- **Previous Track**: Go to the previous track in the playlist
- **Next Track**: Skip to the next track in the playlist
- **Progress Bar**: Click anywhere on the progress bar to seek to that position
- **Volume Slider**: Adjust the playback volume
- **Playlist**: Click on any track in the playlist to play it

## Browser Support

This music player works best in modern browsers that support the HTML5 Audio API:
- Chrome
- Firefox
- Safari
- Edge
- Opera

## Note

This is a client-side only music player. Your music files are not uploaded to any server - they are played directly from your computer using the browser's built-in audio capabilities.
